# GOLDBRIX ONE

Official non-custodial Android wallet for the GOLDBRIX network.

## Download

- Latest public APK release: https://github.com/richlifeguideonly-prog/goldbrix-one/releases/latest

## Live Network Links

- Explorer: https://89-167-36-203.sslip.io/explorer/
- Core repository: https://github.com/richlifeguideonly-prog/goldbrix-core
- Explorer repository: https://github.com/richlifeguideonly-prog/goldbrix-explorer

## What GOLDBRIX ONE Does

- create or import a wallet
- use native `bn1` addresses
- receive funds
- monitor confirmed on-chain balance
- broadcast real transactions to GOLDBRIX mainnet

## Install

1. Open the latest public release page.
2. Download the official APK asset.
3. Allow installation from unknown sources if Android asks.
4. Install and open GOLDBRIX ONE.

## Network Model

- self-custody
- public proof-of-work
- no admin keys
- no founder allocation
- user-controlled wallet access
